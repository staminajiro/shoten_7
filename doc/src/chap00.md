# はじめに

IoT(Internet of Things)を支えるユニークで高性能な様々なデバイスが誕生しています。また、深層学習などを活用することにより
人間と同じくらいの画像認識力を機械に実装することができるようになりました。
「最新デバイスで作る！遊びのレシピ集」は最新デバイスやテクノロジーを使ってちょっと懐かしくてオーバースペックな遊び作りに取り組んだ内容をまとめたものです。
